# Roadmap de desarrollo

## Completado en esta base

- Solucion .NET y proyectos por capas
- Swagger, JWT y middleware global de errores
- Modelo principal de dominio
- EF Core con configuraciones y migracion inicial
- Modulos base de auth, business, categories, products, inventory, sales y reports
- Docker Compose para SQL Server local
- Pruebas unitarias e integracion representativas

## Siguiente iteracion recomendada

- ampliar pruebas para mas reglas de inventario y reportes
- endurecer respuestas de validacion con payload estandar por campo
- agregar seeds de demostracion no productivos
- agregar ejemplos request/response por endpoint
- documentar diagrama entidad-relacion para portafolio
